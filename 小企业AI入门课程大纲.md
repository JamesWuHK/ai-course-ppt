# 小企业AI入门实战课
## 从0基础到智能经营 — 2小时快速上手

---

## 📋 课程概述

**目标学员**：小企业老板、个体经营者、创业者  
**课程时长**：2小时（120分钟）  
**课程目标**：让零基础学员理解AI、掌握实用工具、学会应用到实际经营中

---

## ⏱️ 课程时间安排

### 第一阶段：AI认知破冰（20分钟）

#### 1.1 什么是AI？（8分钟）

**【讲师开场话术】**
> "各位老板，先问大家一个问题：你们手机里有没有用导航软件？导航就是最早的AI——它学习了几百万条路，告诉你怎么走最快。今天的AI，就是更聪明的'导航'，只不过它导的不只是路，还有你的生意。"

**核心概念讲解（5分钟）**

1. **AI不是魔法，是工具**
   - 类比：AI就像计算器，不会替你算账，但能让算账快100倍
   - 关键认知：AI不会取代老板，但会用AI的老板会取代不会用的

2. **大白话解释：AI = 会学习的智能助手**
   - 传统软件：你告诉它1+1=2，它永远只会1+1=2
   - AI：你给它看100道1+1，它自己学会了加法，还能做减法
   - 生意语言：就像一个实习生，刚开始什么都不会，但学得飞快

3. **为什么现在要学？—— 3个信号（3分钟）**
   - **信号1：客户变了** —— 00后消费者期待24小时响应，你做不到AI做得到
   - **信号2：对手变了** —— 隔壁店用AI写文案、做客服，成本比你低一半
   - **信号3：工具变了** —— 以前AI需要程序员，现在会打字就能用

**案例冲击（2分钟）**
- 展示：某小餐馆用AI客服，一个月省了3000元人工成本
- 展示：某个体户用AI写小红书，粉丝从100涨到1万
- 提问："这两个案例，哪个离你更近？"

#### 1.2 AI能帮我做什么？（7分钟）

**【互动开场】**
> "大家把最头疼的3件事写在纸上，我赌至少2件AI能帮你解决。"

**AI能力地图（5分钟）**

用一张图展示AI的5大能力象限：

| 能力 | 具体场景 | 节省成本 |
|------|----------|----------|
| **写** | 朋友圈文案、产品描述、回复客户 | 1个文案月薪5000元 |
| **说** | 自动接电话、回复微信、发语音 | 1个客服月薪4000元 |
| **看** | 看报表、看图片、看客户评价 | 节省每天2小时 |
| **想** | 分析数据、出主意、做计划 | 相当于免费顾问 |
| **做** | 生成图片、做视频、做PPT | 省去外包费用 |

**真实案例展示（2分钟）**

*案例1：餐饮店张老板*
- 痛点：每天花2小时回微信，经常漏单
- 解决方案：AI客服自动回复常见问题
- 效果：客户满意度提升，自己专注做菜

*案例2：服装店李姐*
- 痛点：不会写文案，请人要花钱
- 解决方案：AI每天生成3条朋友圈+5个小红书标题
- 效果：粉丝增长3倍，咨询量翻倍

*案例3：培训机构王校长*
- 痛点：做PPT课件太耗时
- 解决方案：AI一键生成课件大纲+配图建议
- 效果：1小时工作量变成10分钟

**【过渡话术】**
> "看到这里，可能有老板想：这么多工具，我得学多久？答案是——不需要学一堆工具，接下来我会告诉你一个秘密..."

#### 1.3 常见误区澄清（5分钟）

**【讲师话术】**
> "在正式开始之前，我必须先帮大家破除4个心魔。这些想法会阻碍你赚钱，必须现在清除。"

**误区1：AI会取代我（1分钟）**
- ❌ 错误想法："用了AI，我是不是就没用了？"
- ✅ 正确认知：AI是放大器，不是替代者
- 类比：用计算器的人没取代会计，而是比不会用计算器的会计强10倍
- 金句："取代你的不是AI，是会用AI的竞争对手"

**误区2：需要懂编程（1分钟）**
- ❌ 错误想法："我初中文凭，技术搞不来"
- ✅ 正确认知：会打字、会说话就能用
- 现场演示：打开DeepSeek，像微信聊天一样发消息
- 强调："你不需要知道发动机原理，会踩油门就能开车"

**误区3：要花很多钱（1.5分钟）**
- ❌ 错误想法："AI是大公司用的，我们小本生意用不起"
- ✅ 正确认知：基础功能免费，进阶功能每月几十块
- 价格对比表：
  - 请一个文案：5000元/月
  - 请一个客服：4000元/月
  - AI工具：0-100元/月
- 算账："用AI一个月省8000，一年省10万"

**误区4：效果不如人工（1.5分钟）**
- ❌ 错误想法："机器写的能有感情吗？"
- ✅ 正确认知：80%的标准工作，AI比人快、比人稳
- 承认局限：创意策划、深度沟通还需要人
- 定位清晰："AI做重复劳动，你去做更有价值的事"

**【互动环节】**
> "还有谁有顾虑？现在提出来，我们现场解决。"
- 预留2分钟现场答疑

### 第二阶段：AI基础速览（15分钟）

#### 2.1 一个AI工具走天下（8分钟）

**【讲师话术】**
> "好，现在进入实战环节。我要告诉大家一个秘密——你们不需要学10个工具，只需要学会和1个AI对话。"

**为什么只需要一个入口？（2分钟）**

*展示对比图：*
```
传统方式：        AI数字员工方式：
写文案 → 找文案工具    一句话 → AI全搞定
做表格 → 打开Excel     一句话 → AI全搞定
回客户 → 复制粘贴      一句话 → AI全搞定
做分析 → 请教专家      一句话 → AI全搞定
```

**核心认知转变**：
- 以前：人学习工具 → 工具帮你做事
- 现在：人说话 → AI理解 → AI调用工具 → 出结果
- 你不需要知道工具在哪，只需要知道你要什么

**现场演示（4分钟）**

*场景设定：假设我是一家奶茶店老板*

**任务1：写朋友圈文案**
- 输入："我是奶茶店，新上了芋泥波波奶茶，帮我写3条朋友圈文案，要吸引年轻人"
- 展示AI输出结果
- 讲解：AI自动理解了"吸引年轻人"=用网络热梗+表情符号

**任务2：分析销售数据**
- 输入："上周一到周日，每天销量分别是120杯、135杯、98杯、142杯、156杯、189杯、175杯，分析一下"
- 展示AI输出：自动计算平均值、找出最高最低、分析周末效应
- 讲解：不需要懂Excel公式，说话就行

**任务3：生成海报描述**
- 输入："我要做一张芋泥波波奶茶的海报，给我描述一下画面，要发给设计师"
- 展示AI输出：详细的视觉描述、配色建议、元素清单

**【学员跟练】（2分钟）**
> "现在轮到你们。拿出手机，打开微信搜索'DeepSeek'，输入你的行业+一个需求。"
- 示例提示："我是开花店的，帮我写一条情人节促销文案"
- 讲师巡视，现场指导
- 邀请1-2位学员分享结果

#### 2.2 提示词速成（7分钟）

**【讲师话术】**
> "刚才大家体验了一下，有人觉得AI很神，有人觉得AI很傻。区别在哪？——提示词。提示词就是和AI说话的技巧，学会这个，AI能力翻10倍。"

**万能公式：RTGF法则（2分钟）**

```
R - Role（角色）：你是谁？AI是谁？
T - Task（任务）：要做什么？
G - Goal（目标）：要达到什么效果？
F - Format（格式）：以什么形式输出？
```

**对比演示：差提示词 vs 好提示词（3分钟）**

*场景：让AI写一条促销文案*

❌ **差提示词**：
```
帮我写个促销文案
```
- AI输出：通用模板，毫无吸引力

✅ **好提示词**：
```
【角色】你是一位有10年经验的餐饮营销专家
【任务】为我的火锅店写一条周年庆促销文案
【目标】吸引25-35岁的年轻上班族来聚餐，强调性价比和氛围
【格式】3条不同风格的文案，每条配3个合适的话题标签
```
- AI输出：精准、有针对性、可直接使用

**现场练习（2分钟）**
> "拿出刚才写的需求，按照RTGF公式改写。"
- 学员改写自己的提示词
- 对比前后效果
- 讲师点评2-3个案例

**【赠送】提示词模板卡**
- 现场发放/展示：5个常用场景的提示词模板
  - 写文案模板
  - 分析数据模板
  - 回复客户模板
  - 做计划模板
  - 想创意模板

### 第三阶段：热门应用实战（70分钟）

#### 3.1 "养龙虾"模式深度解析（25分钟）

**【讲师开场】（1分钟）**
> "接下来这个环节，可能会改变你们对营销的认知。我要讲的是2025年最火的小企业获客玩法——'养龙虾'。"

**什么是"养龙虾"？（4分钟）**

*现象级案例展示：*
- 小红书截图：某博主"养了一只AI龙虾"，每天喂食、聊天、发朋友圈
- 数据：相关话题阅读量破10亿，衍生出"养螃蟹""养水母"等变体

*本质拆解：*
```
养龙虾 = 虚拟陪伴 + 情感投射 + 社交货币

虚拟陪伴：现代人孤独，需要"被需要"的感觉
情感投射：把龙虾当成宠物，投入感情
社交货币：晒龙虾=有品位、跟潮流、会玩
```

*为什么火？3个底层逻辑：*
1. **零成本**：不需要真的养，AI生成图片+对话
2. **高互动**：每天喂食、聊天，形成习惯
3. **强传播**：晒图自带话题，引发模仿

**商业逻辑拆解（8分钟）**

*1. 情感经济（3分钟）*
- 马斯洛需求：生理→安全→社交→尊重→自我实现
- 养龙虾击中的是"社交需求"+"情感需求"
- 生意启发：顾客买的不是产品，是情感体验

*案例：为什么盲盒火？*
- 不是买玩具，是买"期待感"和"惊喜感"
- 养龙虾同理：不是养宠物，是买"陪伴感"和"成就感"

*2. 社交裂变（3分钟）*
- 传播公式：有趣 + 低门槛 + 可炫耀 = 病毒传播
- 养龙虾的分享点：
  - "看，我的龙虾今天长大了"（成就感）
  - "我的龙虾会说话了"（新奇感）
  - "快来帮我喂龙虾"（互动感）

*生意启发：设计可传播的"社交货币"*

*3. 私域沉淀（2分钟）*
- 从"养龙虾"到"加微信"的路径：
  ```
  看到别人的龙虾 → 好奇 → 询问怎么养 → 加微信获取攻略 → 进入私域
  ```
- 关键：用免费/有趣的内容，换取用户的联系方式

**小企业实战案例（8分钟）**

*案例1：火锅店"养小龙虾"会员体系*
- 玩法：顾客扫码领养一只"虚拟小龙虾"，每天签到"喂食"
- 成长机制：连续喂食7天，小龙虾长大，兑换7折券
- 裂变设计：分享龙虾成长图到朋友圈，获得额外饲料
- 效果：会员活跃度提升60%，朋友圈曝光增长5倍

*案例2：宠物店AI宠物顾问*
- 玩法：顾客上传自家宠物照片，AI生成"宠物日记"
- 内容：每天自动生成宠物的"心情""想吃的东西""想玩什么"
- 转化：日记里自然植入产品推荐（"今天主子想吃罐头"）
- 效果：顾客每天主动来看，复购率提升40%

*案例3：茶叶店"云养茶树"*
- 玩法：顾客认养一棵茶树，通过小程序看茶树生长
- 互动：浇水、施肥（积分兑换），成熟后收获专属茶叶
- 社交：茶树证书、收获晒图、茶树故事
- 效果：客单价从200元提升到800元（认养套餐）

*案例4：书店"养书虫"阅读打卡*
- 玩法：读者领养一只"书虫"，每天阅读打卡"喂食"
- 成长：读得越多，书虫越大，解锁专属书单和徽章
- 社区：书虫排行榜、书虫配对（找书友）
- 效果：会员阅读时长提升3倍，社群活跃度大增

*案例5：健身房"养肌肉"*
- 玩法：会员领养一个"虚拟肌肉小人"，锻炼让它长大
- 联动：同步运动数据，练得越多小人越壮
- 激励：小人达到一定等级，兑换私教课/周边
- 效果：会员到课率从30%提升到65%

**学员实操（4分钟）**
> "现在，用我们刚才学的RTGF提示词，设计一个你自己行业的'养成'玩法。"

*任务要求：*
- 给养成对象起个名字
- 设计3个互动动作
- 设计1个裂变机制
- 设计1个转化路径

*讲师指导：*

请帮我设计：
1. 养成对象是什么？叫什么名字？
2. 顾客每天可以做什么互动？
3. 如何激励顾客分享传播？
4. 如何引导到实际消费？
```
- 学员现场生成方案
- 邀请2-3位学员分享，讲师点评优化

#### 3.2 数字员工搭建实战（35分钟）

**【讲师开场】（1分钟）**
> "如果说'养龙虾'是帮你获客，那'数字员工'就是帮你省钱。接下来这35分钟，我会手把手教你们搭建一个24小时在线、不要工资、不会辞职的超级员工。"

**什么是数字员工？（4分钟）**

*概念澄清：*
- 不是简单的自动回复机器人
- 是具备"理解-思考-执行"能力的AI员工
- 可以学习你的业务知识，像老员工一样服务客户

*能力对比表：*
| 能力 | 人工员工 | 数字员工 |
|------|----------|----------|
| 工作时间 | 8小时 | 24小时 |
| 响应速度 | 几分钟 | 秒回 |
| 知识储备 | 靠记忆 | 无限存储 |
| 情绪状态 | 会累会烦 | 永远耐心 |
| 工资成本 | 4000+/月 | 0-100/月 |
| 离职风险 | 有 | 无 |

*数字员工能做什么？*
- 前台：接待客户、回答问题、引导下单
- 销售：推荐产品、跟进客户、促成转化
- 运营：写文案、做分析、管数据
- 培训：带新人、答疑问、传知识

**搭建你的第一个数字员工（22分钟）**

*【Step 1: 选择平台】（3分钟）*

**推荐平台对比：**

| 平台 | 优势 | 适合谁 | 费用 |
|------|------|--------|------|
| **扣子(Coze)** | 功能最全，插件丰富，可视化搭建 | 想深度使用的 | 免费版够用 |
| **文心智能体** | 百度搜索整合，流量入口 | 依赖百度生态的 | 免费 |
| **钉钉魔法棒** | 与钉钉无缝衔接 | 已用钉钉办公的 | 免费 |
| **腾讯元器** | 微信生态整合 | 依赖微信的 | 免费 |

*讲师建议：*
> "第一次搭建，推荐用'扣子'。功能最全，教程最多，而且免费版就能满足大部分需求。"

*【Step 2: 设计员工人设】（5分钟）*

**人设设计四要素：**

1. **名字**：好记、有行业关联、有亲切感
   - 示例：餐饮店→"小味"、花店→"小花"、书店→"小书童"

2. **性格**：符合品牌调性
   - 高端店→专业优雅、快餐店→活泼热情、专业店→严谨可靠

3. **能力边界**：明确能做什么、不能做什么
   - 能做：回答产品问题、推荐套餐、处理常见投诉
   - 不能做：处理退款（转人工）、处理复杂纠纷（转人工）

4. **说话风格**：统一的语言习惯
   - 示例：每句话带表情、用网络热词、称呼客户"宝子"

**现场演示：创建"小美"——奶茶店智能助手**
```
【名字】小美
【性格】热情活泼，像闺蜜一样亲切
【能力】懂所有产品、会推荐搭配、能处理常见咨询
【风格】用emoji、说网络语、偶尔玩梗
【边界】复杂客诉转人工，涉及退款转人工
```

*学员练习：*
> "给你的数字员工起个名字，写一段人设描述。"

*【Step 3: 配置核心能力】（12分钟）*

**3.1 知识库配置（4分钟）**

*什么是知识库？*
- 数字员工的"大脑"，存储所有业务知识
- 客户问什么，从这里找答案

*知识库内容清单：*
```
📁 产品知识

**3.2 工作流设计（5分钟）**

*典型客服工作流：*
```
客户提问
    ↓
AI理解意图（是什么问题？）
    ↓
查询知识库（找答案）
    ↓
生成回复（组织语言）
    ↓
判断：是否解决？
    ↓ 是          ↓ 否
  结束对话    转人工客服
```

*进阶工作流示例：*
```
客户问"有什么推荐"
    ↓
AI询问："您几个人？喜欢什么口味？预算多少？"
    ↓
根据回答推荐套餐
    ↓
询问："需要现在下单吗？"
    ↓ 是          ↓ 否
  发送下单链接   发送优惠券"下次再来"
```

*现场演示：*
- 在扣子中创建工作流
- 添加"用户输入"节点
- 添加"知识库查询"节点
- 添加"回复生成"节点
- 添加"条件判断"节点

*演示：*

**测试清单：**

**数字员工进阶玩法（6分钟）**

*玩法1：销售助手*
- 自动识别客户意向
- 推送个性化产品推荐
- 跟进未成交客户
- 案例：某服装店用AI销售助手，转化率提升25%

*玩法2：数据分析师*
- 问一句就出报表
- 示例："这个月哪个产品卖得最好？"
- 示例："客户最常在什么时间下单？"
- 示例："预测下周需要备多少货？"

*玩法3：内容运营*
- 每天早上自动生成今日朋友圈文案
- 根据热点事件生成相关内容
- 自动回复评论和私信

*玩法4：培训导师*
- 新员工24小时问答
- 产品知识随问随答
- 模拟客户场景训练

*玩法5：智能预约*
- 客户说"我想预约"→AI询问时间→确认→发送预约确认
- 自动发送提醒消息
- 处理改期、取消

#### 4.1 选择你的第一个AI项目（8分钟）

**决策框架（3分钟）**

*自测三连问：*

**问题1：最耗时的重复性工作是什么？**
- 每天花2小时回微信？→ 优先做数字员工
- 每天想文案想到头秃？→ 优先用AI写文案
- 每天处理重复咨询？→ 优先做AI客服

**问题2：客户最常问的问题有哪些？**
- 如果80%问题都是重复的→数字员工ROI最高
- 如果客户总是问"有什么推荐"→AI导购效果好

**问题3：你最不擅长但必须要做的事？**
- 不擅长写文案→用AI补齐短板
- 不擅长数据分析→用AI做报表
- 不擅长社交运营→用"养龙虾"模式

**推荐起步项目**（二选一）：

*路径A：养龙虾模式（适合获客引流）*
- **适合谁**：缺客户、想扩大知名度、愿意尝试新玩法
- **投入成本**：低（主要是创意和时间）
- **见效周期**：1-2周
- **关键成功因素**：
  - 养成对象要有趣、有记忆点
  - 分享机制要简单、有面子
  - 转化路径要自然、无压力

*路径B：数字员工模式（适合降本增效）*
- **适合谁**：已有稳定客户、人手紧张、重复工作多
- **投入成本**：中（需要配置知识库）
- **见效周期**：1周内
- **关键成功因素**：
  - 知识库要全、要准
  - 人设要符合品牌
  - 转人工机制要顺畅
