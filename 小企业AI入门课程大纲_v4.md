# 小企业AI入门实战课
## 从0基础到AI Agent智能经营 — 2小时快速上手

---

## 🎬 开场悬念（2分钟）

**【讲师话术】**
> "在正式开始之前，我先问大家一个问题——"
> 
> "你们店里有没有那种事情，每天都要做，明明很烦，但又不得不做？比如每天发朋友圈、算账、回客户问同样一个问题..."
> 
> "好，我告诉你一个秘密——这些事情，以后你只需要说一句话，AI帮你全干了。"

**⚡ 钩子悬念**：
- "今天我会让你看到，一个从来没有学过AI的人，怎么在10分钟内，用一句话做完平时要花2小时的事。"
- "而且我专门挑了一个你们的同行现场做给你们看——不是模拟，是真做。"
- "课上到一半，我会问一个问题，答对的同学可以加我微信领一份神秘礼物。"

**目的**：**开场30秒内抓住注意力**，用痛点+悬念让人不想走开

---

## 📋 课程概述

**目标学员**：小企业老板、个体经营者、创业者（**0基础、 AI认知程度未知**）  
**课程时长**：2小时（120分钟）  
**课程目标**：理解AI Agent时代的工作方式，学会用自然语言驱动AI完成经营任务

**⚠️ 课程设计关键前提**：
- 学员是**完全0基础**的小企业主，可能从未用过AI工具
- 学员是**陌生客户**，不知道他们的AI认知程度
- **必须从最基础的AI认知开始讲**，不能假设任何先验知识
- **全程互动**，随时了解学员反馈和理解程度

**核心转变**：
- 旧思维：人操作软件 → 软件执行任务
- 新思维：人描述需求 → AI Agent自动调用工具 → 完成任务

#### 1.0 AI基础概念小百科（7分钟）

**⚠️ 这一节照顾完全0基础学员，快速过一遍，有基础的可以加速理解**

**【讲师话术】**
> "正式开始之前，我先快速帮大家补一课——最近大家是不是经常听到'AI'、'Agent'、'豆包'这些词？到底都是什么？用最通俗的话解释一下。"

**什么是AI（人工智能）？**
- "你可以理解为：一个特别聪明的虚拟大脑"
- "它读过全世界所有的书、看过所有的视频、听过所有的对话"
- "所以你问它什么问题，它都能回答你"
- "就像一个全世界知识最渊博的朋友，24小时随时在线，还不要工资"

**你可能用过的AI产品（接地气版）**：

| 产品名 | 像什么 | 能做什么 |
|--------|--------|----------|
| **豆包** | 智能小助手 | 聊天、写作、解答问题（像 Siri/小爱同学升级版） |
| **Kimi** | 文书助手 | 读长文章、总结文档（像帮你看合同的秘书） |
| **通义千问** | 阿里版ChatGPT | 聊天、推理、解决问题 |
| **ChatGPT** | 国外最火的AI | 聊天、写作、编程（要翻墙） |
| **文心一言** | 百度版ChatGPT | 聊天、画画、写作 |

**【讲师话术】**
> "你们用过其中任何一个吗？没用过也没关系，你就理解为——这些是'会聊天的超级知识库'。你问它答，像和一个很聪明的人聊天。"

**🤔 AI 和 Agent 有什么区别？**

| | AI（人工智能） | Agent（智能代理） |
|--|----------------|-------------------|
| **像什么** | 一个聪明的**人** | 一个能干的**员工** |
| **会做什么** | 回答问题、写东西 | 帮你**执行任务** |
| **例子** | 你问"怎么做红烧肉"，它告诉你 | 你说"帮我做红烧肉"，它帮你做 |
| **比喻** | 顾问军师 | 执行者管家 |

**【讲师话术】**
> "简单说：AI是'能说'，Agent是'能做'。"
> 
> "你们就把Agent理解成一个——你给它下命令，它帮你干活的那种员工。"
> 
> "咱们今天要学的，就是怎么让这个'AI员工'帮你经营生意。"

**📱 什么是"养龙虾"？**

> "最近科技圈有个热词叫'养龙虾'，很多人以为是真的养虾——不是！"
>
> "**养龙虾**是指学习和使用**OpenClaw**这个开源AI工具。"
>
> "为什么叫'养龙虾'？因为OpenClaw的Logo是一只红色小龙虾，'Claw'是爪子的意思。网友觉得学习用这个工具像'养虾'一样有意思，就把它叫'养龙虾'了。"
>
> "**OpenClaw是什么？** 简单说就是一个能'动手干活'的AI——它不只是回答问题，它能直接帮你操作电脑、处理文件、发消息、管日程。"
>
> "这就相当于一个**数字员工**，你让它干什么，它就帮你干。"

**【讲师话术】**
> "养龙虾的本质，就是学会用AI工具帮你干活。学会了，你就有了一个24小时不睡觉、不要工资、不会辞职的员工。"

**目的**：纠正错误理解，建立正确认知：养龙虾=学会用OpenClaw这个AI工具

**✅ 本节收尾**
> "好，基础概念补完了。有没有谁对这些词还是不太清楚的？评论区打'不懂'，我再说一遍。"
> 
> "都懂了？那接下来放大招——让你们看看这个'AI员工'到底能帮你干什么。"

**软件时代的痛点（3分钟）**

*传统工作方式：*
```
写文案 → 打开Word → 打字 → 修改 → 保存
做图 → 打开PS/Canva → 找模板 → 改图 → 导出
回客户 → 打开微信 → 打字 → 复制粘贴 → 发送
分析数据 → 打开Excel → 输公式 → 做图表 → 截图
```

*问题：*
- 要学N个软件的操作
- 每个软件功能只用10%
- 人在软件之间来回切换
- 重复劳动浪费大量时间

**Agent时代的到来（4分钟）**

*什么是AI Agent？*
- 不是单个工具，是"智能管家"
- 你描述需求，Agent自动选择工具、执行任务
- 背后可以调用几十个Skill（技能）

*类比理解：*
```
传统方式：你自己当司机，学开车、认路、加油
Agent方式：你说"去机场"，AI司机自动规划路线、开车、找停车位
```

*WorkBuddy的Skill生态示例：*
- 飞书Skill：操控飞书文档、多维表、日历、IM、会议...整个飞书办公套件
- 钉钉Skill：AI表格、日历、待办、DING消息、通讯录等（钉钉CLI已开源）
- 企微Skill：消息、日程、文档、智能表、会议、待办、通讯录等（企微CLI今日开源）
- 海报Skill：输入需求直接生成可用海报，不需要描述画面
- 数据分析Skill：读取数据、生成图表、写分析报告
- 图像/视频Skill：生成产品图、短视频
- 搜索Skill：查资料、查价格、查竞品

> **💡 趋势备注（2026年3月，就在这几天）**：飞书、钉钉、企业微信三大企业IM平台**同周**相继开源CLI工具——这不是巧合，这是整个企业软件生态在向"Agent友好"转型的集体宣示。你用的是哪个平台，Agent都能直接操控。

**现场演示（3分钟）**

*场景：假设我是奶茶店老板*

**传统方式：**
- 打开飞书表格看销售数据
- 手动汇总，写报告
- 打开飞书IM发给合伙人
- 每个步骤都要人操作

**Agent方式：**
- 对WorkBuddy说："分析一下上周飞书表格里的销售数据，生成一份分析报告，通过飞书发给合伙人"
- Agent自动：
  1. 调用飞书Skill读取多维表数据
  2. 调用分析Skill计算统计
  3. 调用飞书Skill创建文档写入报告
  4. 调用飞书Skill发送IM消息

**关键认知转变：**
- 不需要手动操作飞书，Agent帮你点击、填写、发送
- 不需要知道海报怎么做，直接说"做一张海报"
- 不需要在各个界面来回切换，Agent自动协调

#### 1.2 为什么小企业更需要Agent？（8分钟）

**【互动开场】**
> "大家算过账吗？如果请一个会Excel+PS+文案+客服的全能员工，一个月要多少钱？"

| 你需要 | 传统方案 | Agent方案 |
|--------|----------|-----------|

- 不会累、不会烦、不会辞职

### 第二阶段：Agent工作方式实战（40分钟）

#### 2.1 用自然语言驱动工作（15分钟）

**【讲师话术】**
> "Agent时代，最重要的技能不是学软件，而是学会'说话'。你描述得越清楚，Agent执行得越到位。"

**自然语言指令的RTGF法则（5分钟）**

*公式：*
```
R - Role（角色）：你希望Agent扮演什么角色？
T - Task（任务）：具体要做什么？
G - Goal（目标）：要达到什么效果？
F - Format（格式）：以什么形式输出？
```

*对比示例：*

❌ **差指令**：
```
帮我写个文案
```
- Agent困惑：写什么文案？给谁看？什么风格？
- 输出：通用模板，基本不能用

✅ **好指令**：
```
【角色】你是一位有10年经验的餐饮营销专家
【任务】为我的火锅店写一条周年庆促销文案
【目标】吸引25-35岁的年轻上班族来聚餐，强调性价比和氛围
【格式】3条不同风格的文案，每条配3个合适的话题标签
```
- Agent清晰：知道角色、任务、目标、格式
- 输出：精准、可直接使用

**🗣️ 现场互动：你来说，我来做（8分钟）**

*【邀请一位学员参与】*

> "张老板，你平时最头疼的事是什么？告诉我一件，我现场用Agent帮你做。"

**操作流程**：
1. 学员说出日常困扰（如"写文案太慢"、"算账麻烦"）
2. 讲师现场演示Agent执行
3. 学员现场看效果
4. **让学员打分**：满意 / 一般 / 不满意

*场景：奶茶店老板的早晨*

**任务1：看昨天数据（2分钟）**
- 指令："看一下飞书多维表里昨天各产品的销量，告诉我卖得最好和最差的是什么，分析一下可能的原因"
- Agent执行：
  - 调用飞书Skill直接读取多维表数据（无需人手动导出）
  - 调用分析Skill计算排序
  - 生成自然语言报告
- 输出："昨天芋泥波波销量最高（156杯），可能是新品效应；经典奶茶销量下降20%，可能是天气变热..."

**任务2：写今日朋友圈（2分钟）**
- 指令："今天周三，帮我写3条朋友圈文案推芋泥波波，要吸引上班族下午茶，风格轻松活泼"
- Agent执行：
  - 调用文案生成Skill
  - 结合周三、下午茶场景
- 输出：3条带emoji的文案，可直接发

**任务3：回复客户咨询（2分钟）**
- 指令："有个客户问'你们家奶茶甜不甜，有没有推荐的'，帮我回复"
- Agent执行：
  - 调用知识库Skill读取产品信息
  - 生成个性化推荐
- 输出："我们家甜度可以选哦~推荐您试试芋泥波波，三分甜刚刚好..."

**任务4：做一张促销海报（2分钟）**
- 指令："做一张'周三会员日 第二杯半价'的海报，清新风格，奶茶店氛围"
- Agent执行：
  - 调用海报Skill，直接生成可用海报
  - **不需要描述颜色、布局、元素，Skill自动处理**
- 输出：可直接转发的促销海报图

**【讲师说明】**
> "注意——以前做海报你要么学PS要么找外包，现在一句话10秒出图。而且觉得不满意，直接说'换成红色调''再突出一下折扣'，Agent继续改。"

**关键洞察（2分钟）**

*你不需要知道：*
- 飞书哪个功能在哪里
- 海报该怎么排版
- 数据该用什么公式

*你只需要知道：*
- 你要什么
- 给谁看
- 达到什么目的

*Agent自动处理：*
- 选择什么工具
- 怎么操作工具
- 怎么组合输出

#### 2.2 Skill生态介绍（12分钟）

**【讲师话术】**
> "Agent之所以强大，是因为背后有Skill生态。就像手机有App Store，Agent有Skill Store。"

**什么是Skill？（3分钟）**

*定义：*
- Skill = AI能用的"超能力"
- 每个Skill封装了一个工具或能力
- Agent根据你的需求，自动调用合适的Skill

*类比：*
```
Agent = 智能手机
Skill = App
你描述需求 = 说"我想导航"
Agent自动 = 打开地图App
```

#### 2.3 与Agent协作的最佳实践（13分钟）

**【讲师话术】**
> "Agent很强大，但要用好也需要技巧。接下来分享几个实战技巧。"

**技巧1：从简单任务开始（3分钟）**

*新手常见错误：*
- 一上来就想让Agent做复杂项目
- 指令模糊，期望Agent猜到你的意思
- 不做反馈，一次不满意就放弃

*正确姿势：*
```
第1周：用Agent做单一任务
  - "写一条朋友圈文案"
  - "算一下今天的营业额"
  
第2周：组合2-3个任务
  - "写文案+配图"
  - "分析数据+生成报告"
  
第3周：尝试复杂项目
  - "做一个完整的促销方案"
  - "搭建一个自动客服"
```

**技巧2：迭代优化（4分钟）**

*Agent不是一次到位的：*
- 第一次输出可能不完美
- 通过反馈让Agent调整
- 多轮对话打磨结果

*示例：写文案的迭代*

**Round 1：**
- 你："写一条新品文案"
- Agent：生成通用文案

**Round 2：**
- 你："太正式了，要更活泼一些，加一些emoji"
- Agent：调整语气风格

**Round 3：**
- 你："再加一个限时优惠的紧迫感"
- Agent：加入促销元素

**Round 4：**
- 你："完美，再生成2个不同版本的"
- Agent：输出多个选项

*关键：*
- 把Agent当实习生，耐心指导
- 每次给出具体反馈
- 保存好的指令模板，下次复用

**技巧3：建立个人知识库（3分钟）**

*什么是个人知识库？*
- 你的业务信息：产品、价格、政策
- 你的品牌风格：语气、调性、禁忌
- 你的常用模板：文案、话术、方案

*为什么要建？*
- 让Agent更懂你的业务
- 输出更符合你的风格
- 减少重复说明

*怎么建？*
```
对Agent说：
"记住以下信息：
- 我的店叫'小味火锅'，主打川味
- 我的客户主要是25-35岁上班族
- 我的风格是热情、接地气、带网络梗
- 我的价格区间是人均80-120元
以后写文案都按这个风格来"
```

**技巧4：人机协作（3分钟）**

*Agent擅长的：*
- 重复性工作
- 数据分析
- 快速生成初稿
- 24小时响应

*人擅长的：*
- 创意决策
- 情感沟通
- 复杂判断
- 关系维护

*最佳分工：*
```
Agent做80%：生成、整理、回复、计算
人做20%：把关、调整、关键决策、深度沟通
```

*示例：客户投诉处理*
- Agent：第一时间响应、安抚情绪、记录问题
- 人：复杂情况介入、情感沟通、最终解决方案

## 💡 讲师备注

### 教学技巧
1. **多演示少理论**：每讲5分钟理论，演示5分钟实操
2. **用学员语言**：避免技术术语，用生意语言
3. **即时成就感**：让学员在课上就完成第一个Agent任务
4. **消除恐惧**：强调"说话就行，不用学软件"
5. **⚠️ 0基础前提**：不要假设学员有任何AI背景知识
6. **随时互动检查**：每15分钟问一次"有没有不懂的"
7. **根据反馈调速**：如果多数人没听懂，放慢并多举例子

### 🎤 线上演讲技巧（关键！）

**开场30秒法则**：
- 前30秒必须抓住注意力，否则学员会划走
- 用痛点问题开场，不用自我介绍
- 例："你是不是每天要花2小时回客户同样一个问题？"

**悬念设置技巧**：
- 每15分钟设置一个悬念
- 例："接下来我要让你们看一个你们绝对想不到的用法..."
- "这个功能，80%的人都不知道..."
- "刚才那位说'AI不可能帮我算账'的学员，看好了..."

**线上互动策略**：
- **高频提问**：每3-5分钟问一次"听懂的打1，没听懂的打2"
- **选择题互动**：让学员在评论区打数字，比打字更容易
- **神秘感话术**：
  - "接下来我要演示的这个功能，一般人我不告诉他"
  - "这是今天最值钱的技巧，你们记好了"
  - "能不能学会，就看接下来这3分钟"
- **福利诱导**：
  - "答对问题的同学，课后私信我领..."
  - "今天直播间前50名报名的，送..."

**保持注意力技巧**：
- 每10分钟换一个形式：讲→演示→互动→提问→故事
- 永远不要超过15分钟纯讲
- 用"接下来"三个字作为转折点标记
- 语速要快，节奏要紧凑，废话全删

**线下转线上特别注意**：
- 线上学员不会主动发言，要**主动点名**："那位叫XX的学员，你在吗？"
- 评论区是生命线，**随时看、随时回应**
- 准备几个"托"，提前安排好互动配合
- **语速比线下快30%**，因为线上容易走神

### 🔥 演讲情绪曲线设计

```
时间轴：0min → 30min → 60min → 90min → 120min
情绪：  好奇 → 兴奋 → 疲惫 → 惊喜 → 冲动行动
         ↑        ↑        ↑        ↑
       开场    第一阶段  中段    最后转化
       悬念    结束时    小高潮   紧迫感
```

**关键节点**：
- 0-5分钟：好奇心峰值（用最大痛点+悬念）
- 30分钟：第一波疲惫（换形式/互动/小故事）
- 60分钟：注意力低谷（来一个意外惊喜演示）
- 90分钟：最后高潮（限时福利/倒计时）
- 120分钟：行动冲动（立刻扫码/报名）

### 常见学员问题准备
- Q: Agent会不会很难学？
- Q: 我的数据安全吗？
- Q: 需要付费吗？
- Q: Agent能替代人工吗？

- Agent响应慢：准备备用案例
- 学员进度不一：准备分层练习
